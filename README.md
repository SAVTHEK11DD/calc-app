# Pulse Calculator

A simple calculator built with HTML, CSS, and JavaScript.

## Features

- Add, subtract, multiply, divide
- Decimal support
- Clear button

## How to Run

1. Clone this repo
2. Open `index.html` in your browser

## Files

- `index.html` – structure
- `style.css` – styling
- `index.js` – logic
